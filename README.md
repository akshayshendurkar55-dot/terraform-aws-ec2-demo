# Terraform AWS EC2 Demo

This project demonstrates Infrastructure as Code (IaC) using Terraform and AWS.

## Features
- Configure AWS Provider
- Create EC2 Instance using Terraform
- Validate Infrastructure
- Plan Infrastructure
- Apply Infrastructure
- Destroy Infrastructure

## Commands Used

terraform init
terraform validate
terraform plan
terraform apply
terraform destroy

## Technologies
- Terraform
- AWS EC2
- AWS IAM
- AWS CLI

## Result
Successfully created and managed an AWS EC2 instance using Infrastructure as Code (IaC).
